# University of Canberra - Modern Website Concept

This is a modern, sleek, and youth-centric website concept for the University of Canberra, featuring glassmorphism design elements and smooth interactions.

## Features
- **Glassmorphism UI**: Frosted glass effects on cards, navigation, and overlays.
- **Modern Typography**: Using 'Outfit' and 'Plus Jakarta Sans' for a clean, contemporary look.
- **Responsive Design**: Fully adaptable to mobile and desktop screens.
- **Interactive Elements**: Custom cursor, hover effects, and scroll animations.

## Sections Included
- **Hero**: Immersive landing with stats and call-to-action.
- **Stories**: "Instagram-style" stories for campus life.
- **Courses**: Grid layout of degree options.
- **Admission**: Application info and quick form.
- **Faculties**: Slider showcasing different departments.
- **Scholarships**: Information on financial support.
- **Management**: Leadership profiles.

## How to Run
Simply open the `index.html` file in any modern web browser to view the website. No installation required.
