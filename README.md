# AJCE - Amal Jyothi College of Engineering (Modern Website Concept)

This is a modern, sleek, and youth-centric website concept for **Amal Jyothi College of Engineering (AJCE), Kanjirappally**, featuring glassmorphism design elements and smooth interactions.

## Features
- **Glassmorphism UI**: Frosted glass effects on cards, navigation, and overlays.
- **Dynamic Background**: Interactive Three.js based "ColorBends" effect that reacts to mouse movement.
- **Modern Typography**: Using 'Outfit' and 'Plus Jakarta Sans' for a clean, contemporary look.
- **Responsive Design**: Fully adaptable to mobile and desktop screens.
- **Interactive Elements**: Custom cursor, hover effects, and scroll animations.

## Sections Included
- **Hero**: Immersive landing with stats and call-to-action.
- **Stories**: "Instagram-style" stories for campus life at AJCE.
- **Courses**: Grid layout of Engineering and Technology departments.
- **Admission**: Application info for B.Tech, M.Tech, MCA, and PhD.
- **Departments**: Highlights of various academic departments.
- **Scholarships**: Information on merit-based and sports support.
- **Management**: Leadership profiles of the college.

## How to Run
Simply open the `index.html` file in any modern web browser to view the website. No installation or build steps required.

## Technologies Used
- HTML5 / CSS3 (Vanilla)
- Vanilla JavaScript
- Three.js (for the dynamic background)
- Font Awesome (for icons)
- Google Fonts
